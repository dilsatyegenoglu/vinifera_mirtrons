# vinifera_mirtrons

vinifera_mirtrons is an open-source computational pipeline for the in silico
discovery and classification of plant mirtron candidates from genome annotations.

## Features
- Intron extraction from GFF + genome FASTA
- Length and GC filtering
- RNA secondary structure analysis
- Scoring-based classification
- Drosha-free mirtron detection
- Comprehensive statistical summaries
